# step-4-checklists/README.md

## Title

Step 4 — Launch Checklists

## Objective

Собрать практичные чек‑листы для предрелизной проверки MVP:

- Производительность (TTFB/TTI/LCP) — быстрая загрузка на мобильных
- Доступность (WCAG 2.1 AA) — контраст, клавиатурная навигация, формы
- SEO — метаданные, JSON‑LD, sitemap/robots, hreflang
- Безопасность — CSP/HSTS/Turnstile/Rate‑limit/2FA/бэкапы
- Финальный QA — ключевые флоу, регрессии, мониторинг, откат

## How to use

- Проходите чек‑листы сверху вниз.
- Фиксируйте результаты в PR описании.
- Скриншоты Lighthouse/axe/Sentry/Платёжного потока — приложить в артефакты.
